### ‚úÖ **1. –°–∏–º–≤–æ–ª—å–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è `Function` (–∞–±—Å—Ç—Ä–∞–∫—Ç–Ω–∞—è, –±–µ–∑ —Ç–µ–ª–∞)**

–ï—Å–ª–∏ —Ç—ã —Ö–æ—á–µ—à—å –ø—Ä–æ—Å—Ç–æ –∑–∞–¥–∞—Ç—å –æ–±–æ–∑–Ω–∞—á–µ–Ω–∏–µ **f(x)** –±–µ–∑ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è –≤—ã—Ä–∞–∂–µ–Ω–∏—è:

```python
from sympy import symbols, Function

x = symbols('x')
f = Function('f')(x)

print(f.diff(x))  # f'(x)
```

–¢–∞–∫—É—é —Ñ—É–Ω–∫—Ü–∏—é –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –≤ —É—Ä–∞–≤–Ω–µ–Ω–∏—è—Ö –∏ –¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä–æ–≤–∞–Ω–∏–∏, –Ω–æ —É –Ω–µ—ë **–Ω–µ—Ç —Ç–µ–ª–∞** ‚Äî –ø—Ä–æ—Å—Ç–æ –∏–º—è.

### ‚úÖ **2. –û–ø—Ä–µ–¥–µ–ª—ë–Ω–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è (—á–µ—Ä–µ–∑ `Eq`, `Lambda` –∏–ª–∏ –æ–±—ã—á–Ω–æ–µ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ)**

–ï—Å–ª–∏ —Ç—ã —Ö–æ—á–µ—à—å –∑–∞–¥–∞—Ç—å **—Ñ—É–Ω–∫—Ü–∏—é —Å —Ç–µ–ª–æ–º**, –∏—Å–ø–æ–ª—å–∑—É–π:

#### üîπ –ß–µ—Ä–µ–∑ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ:

```python
from sympy import symbols, sin

x = symbols('x')
f = sin(x)**2 + x

print(f.diff(x))  # –ø—Ä–æ–∏–∑–≤–æ–¥–Ω–∞—è –æ—Ç sin(x)**2 + x
```

#### üîπ –ß–µ—Ä–µ–∑ `Lambda`:

```python

print(f(x))         # x**2 + 1
print(f(x).diff(x)) # –ø—Ä–æ–∏–∑–≤–æ–¥–Ω–∞—è
```
